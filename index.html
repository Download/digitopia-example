Some Bandages for the Bleeding Edge

Using Node, Express, Loopback and the fertile Node ecosystem to build full stack web services.

This application scaffolding holds the example code on blog.digitopia.com.

The scaffolding makes use of Amazon's AWS s3 service - you will need an AWS account to run the examples.

To run the scaffolding download and unpack it from github, run npm install and run the app.

AWS_S3_KEY="KEY" AWS_S3_KEY_ID="ID" S3_BUCKET="BUCKET" node .
